```
       ║                           ║                  ║

  ║ MongoDB  ║               ║  LLM Layer ║    ║ Google APIs   ║
  ║ Mongoose ║               ║  10-key    ║    ║ Search + YT   ║
  ║ Schemas  ║               ║  rotation  ║    ╚═══════════════╝
  ╚══════════╝               ╚════════════╝
```

**Design Principles**
- **Feature-Based Architecture** — domain modules (`/ai`, `/curriculum`, `/learning`, `/search`) not role folders
- **RSC-First** — Server Components by default, `"use client"` only when interactivity demands it
- **AI Resilience** — 10-key pool keeps Sensei online under full load
- **Schema-Driven Data** — Mongoose models with strict validation + compound indexing

</details>
